Projects
IBM Attrition Risk Scoring with SHAP Explainability
Built a logistic regression pipeline to identify the top employee flight risks from the IBM HR dataset. Handled class imbalance using balanced class weights. Evaluated model using recall and precision with business cost framing. Applied SHAP analysis at both population and individual employee level to generate targeted intervention recommendations. Key finding: overtime, low income, and distance from home are the primary attrition drivers.
